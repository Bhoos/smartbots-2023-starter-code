# smartbots-2023-starter-code

Starter code for smartbots 2023

## Terminology

### Card Denotation

##### Suit Denotation

| Name    | Id  | Examples     |
| ------- | --- | ------------ |
| Spade   | S   | 1S...8S...KS |
| Diamond | D   | 1D...8D...KD |
| Hearts  | H   | 1H...8H...KH |
| Clubs   | C   | 1C...8C...KC |

##### Rank

| Name  | Id  | Examples       |
| ----- | --- | -------------- |
| King  | K   | KS, KD, KH, KC |
| Queen | Q   | QS, QD, QH, QC |
| Jack  | J   | JS, JD, JH, JC |
| Ten   | T   | TS, TD, TH, TC |
| Nine  | 9   | 9S, 9D, 9H, 9C |
| Eight | 8   | 8S, 8D, 8H, 8C |
| Seven | 7   | 7S, 7D, 7H, 7C |
| Ace   | 1   | 1S, 1D, 1H, 1C |

### Bid Denotation

- 0 -> Pass Bid | No Bid
- 16...28 -> Available Bid
